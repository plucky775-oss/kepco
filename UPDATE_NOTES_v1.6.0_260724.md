# v1.6.0 수정사항

## 핵심 Check Point

- 원본 PDF의 사각형(`■`)과 원형(`○`) 항목을 동일 레벨로 변경했습니다.
- 사각형과 원형의 좌측 여백, 본문 시작 위치, 상태 버튼 위치를 동일하게 맞췄습니다.
- 원본 기호는 구분을 위해 그대로 표시합니다.
- 대시(`-`) 항목만 세부 문장으로 들여쓰기합니다.
- 96종 데이터에서 자동 추론했던 `parentId`, `hierarchy`, `level` 속성을 제거했습니다.
- PDF 출력도 같은 레벨 규칙을 적용하고 기존 PDF 캐시를 새 버전으로 재생성합니다.

## 작업 기본정보

다음 입력란을 화면과 PDF에서 제거했습니다.

- 작업일시
- 작업장소
- 회사·연구소
- 부서
- 작업인원
- 참여 작업자
- 비고

남은 항목은 `작업공종(Code)`, `작업명`, `공정명(과제번호)`, `담당자`, `책임자(PL)`이며, 위에서 아래로 간결하게 재배치했습니다.

## 호환성

- 체크포인트 ID와 순서는 변경하지 않았습니다.
- 기존 임시저장·완료 문서의 응답, 사진, 서명은 그대로 유지됩니다.
- 제거된 입력란의 과거 값은 데이터베이스에 남아 있을 수 있으나 화면과 새 PDF에는 표시되지 않습니다.
