# 한전 분산전원 연계조회

주소 또는 현재 위치를 기준으로 한전 전력데이터개방포털의 분산전원 연계정보를 조회하는 독립형 Vercel 웹앱입니다.

## 배포

1. 이 폴더의 파일을 별도 GitHub 저장소에 업로드합니다.
2. Vercel에서 해당 저장소를 새 프로젝트로 가져옵니다.
3. Vercel 프로젝트의 Environment Variables에 다음 값을 등록합니다.

   - Key: `KEPCO_DGEN_API_KEY`
   - Value: 한전 전력데이터개방포털에서 발급받은 인증키
   - Environments: Production, Preview

4. 저장 후 Redeploy 합니다.

인증키는 `/api/dgen` 서버 함수에서만 읽으며 브라우저 코드에는 포함되지 않습니다.

## 조회 정보

- 변전소명
- 주변압기 번호
- 회선명(DL)
- 변전소·주변압기·배전선로 잔여용량
- 각 설비의 총 연계가능용량과 누적 연계용량

공개된 용량은 사전 검토용이며 실제 접속 가능 여부는 한전의 계통연계 기술검토로 최종 확정됩니다.
