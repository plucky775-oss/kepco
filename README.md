# 작업안전 체크리스트 독립앱

Power TBM과 분리된 독립 실행형 PWA입니다.

## 주요 기능
- 96종 표준공정 검색 및 선택
- 정상 / 이상 / 해당 없음 판정
- 이상 내용, 조치사항, 사진 기록
- 담당자 및 책임자 전자서명
- 자동 임시저장 및 완료 문서 보관
- 캘린더 조회
- PDF 및 JSON 저장
- 오프라인 캐시

## 실행
압축 해제 후 폴더에서 다음 명령을 실행합니다.

```bash
python -m http.server 8080
```

브라우저에서 `http://localhost:8080`에 접속합니다.

## 배포
Vercel, Firebase Hosting, GitHub Pages 등 정적 호스팅에 폴더 전체를 배포할 수 있습니다.

## 주의
현재 문서와 사진은 사용자 기기의 IndexedDB에 저장됩니다. 여러 기기 동기화와 관리자 통합조회는 Firebase 연동 단계에서 추가해야 합니다.
