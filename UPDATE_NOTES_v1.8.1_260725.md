# 작업안전 체크리스트 v1.8.1 업데이트 노트

## 체크리스트 선택 화면

- 체크리스트 카드의 위험요인 미리보기 영역을 제거했습니다.
- 검색창은 `workName`(체크리스트 제목·작업명)만 대상으로 검색합니다. 코드, 분류, 위험요인은 검색 대상에서 제외했습니다.
- 검색창 안내 문구를 `체크리스트 제목 검색`으로 변경했습니다.
- 카드 하단에 구성 개수와 작성 버튼을 한 줄로 배치하고, 작은 화면에서는 버튼이 자동으로 전체 폭으로 배치되도록 반응형 스타일을 조정했습니다.
- 목록 카드의 여백, 그림자, 간격을 줄여 더 단정하고 간결하게 표시합니다.

## 호환성

- 기존 대분류·중분류 필터는 유지합니다.
- 기존 IndexedDB 문서 구조와 체크리스트 데이터 구조는 변경하지 않았습니다.

- 서비스워커 캐시: `safety-checklist-standalone-v1.8.1-r2`
