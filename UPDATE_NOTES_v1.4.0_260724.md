# v1.4.0 업데이트 내용

## 화면 간소화

- 상단의 부제 문구를 제거했습니다.
- 홈 화면의 연계 상태 카드와 장문의 안내 문구를 제거했습니다.
- 홈 메뉴의 설명 문구를 제거하고 메뉴명과 문서 건수만 표시합니다.
- 작성 화면의 분류·원본 페이지 안내와 전자서명 보조 문구를 제거했습니다.
- 완료 문서 화면의 브라우저 저장 안내 문구를 제거했습니다.

## 회의록 연계 제거

- 작업 기본정보의 정보 불러오기 버튼을 제거했습니다.
- 기본정보 하단의 연계 안내와 링크를 제거했습니다.
- 새 체크리스트는 공정명, 회사, 장소, 책임자, 참여자 등을 자동으로 가져오지 않습니다.
- 작업일시는 현재 날짜와 시간으로 시작하며 나머지 항목은 직접 입력합니다.

## 호환성

- 기존 IndexedDB 이름은 유지하므로 배포 업데이트만으로 기존 임시저장·완료 문서는 보존됩니다.
- A4 PDF 생성·저장·공유 방식은 유지됩니다.
- 서비스워커 캐시를 v1.4.0으로 갱신했습니다.
