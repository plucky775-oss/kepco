# 작업안전 체크리스트 v1.4.0 검증 결과

검증일: 2026-07-24

## 정적 검사

- `app.js` 문법 검사 통과
- `components/research-checklist.js` 문법 검사 통과
- `sw.js` 문법 검사 통과
- `manifest.webmanifest` JSON 파싱 통과
- 체크리스트 데이터 JSON 파싱 통과
- 표준공정 데이터 96종 로딩 확인
- 서비스워커의 캐시 버전 및 파일 쿼리 `r5` 일치 확인

## 화면 검사

1366×1024 및 390×844 브라우저 화면에서 다음을 확인했습니다.

- 홈 화면의 회의록 연계 상태 영역 제거
- 상단 부제 문구 숨김
- 홈 메뉴의 장문 설명 제거
- 모바일 홈 화면 가로 넘침 없음
- 작업 기본정보의 정보 불러오기 버튼 제거
- 작업 기본정보 하단의 연계 안내 및 링크 제거
- 작성 화면의 핵심 입력·체크·서명 영역 정상 렌더링

## 연계 차단 검사

브라우저 저장소에 기존 회의록 형식의 테스트 데이터를 넣은 상태에서 새 체크리스트를 만들었습니다.

- 공정명 자동 입력 없음
- 회사·연구소 자동 입력 없음
- 작업장소 자동 입력 없음
- 책임자 자동 입력 없음
- 참여 작업자 자동 입력 없음
- 작업일시는 현재 날짜와 시간으로 정상 초기화

## 기존 데이터 호환성

- IndexedDB 이름과 문서 저장소 이름을 변경하지 않았습니다.
- 기존 임시저장·완료 문서는 업데이트 후에도 같은 저장소에서 조회됩니다.
- 기존 PDF 렌더 버전을 유지해 저장된 PDF를 불필요하게 다시 만들지 않습니다.

## PDF 회귀 검사

로컬 PDF 라이브러리 테스트 대역을 사용해 다음 흐름을 확인했습니다.

- 작성 화면에서 PDF 미리보기 실행
- A4 페이지 분할 함수 실행
- 2페이지 PDF Blob 생성
- 빈 PDF 검사 통과
- 미리보기 창의 Blob URL 전환 확인
- 생성 종료 후 진행 화면 정상 제거

실제 PDF 엔진과 문서 레이아웃 코드는 v1.3.1의 정상 동작 버전을 유지했습니다.
