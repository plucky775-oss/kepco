# 2026-07-24 수정본 적용 안내

이번 수정본은 다음 세 가지 현장 제보를 반영했습니다.

1. 작업일시 입력칸이 지나치게 길어 작업장소 입력칸과 겹침
2. 체크포인트 제목과 하단 버튼이 점검 내용을 가림
3. iPad Safari에서 미리보기는 보이지만 PDF 저장 결과가 백지로 생성됨

## 적용 방법

기존 GitHub 저장소의 파일을 수정본으로 덮어쓴 뒤 커밋하고 푸시합니다.

```bash
git add .
git commit -m "Fix tablet layout and iOS PDF export"
git push
```

Vercel이 GitHub 저장소와 연결되어 있다면 푸시 후 자동으로 재배포됩니다. GitHub Pages를 사용 중이라면 Pages 배포가 완료될 때까지 잠시 기다린 뒤 새로고침합니다.

## iPad에서 확인할 항목

- 작업일시와 작업장소 입력칸이 서로 겹치지 않는지
- 핵심 Check Point의 첫 번째 문장이 제목 아래에서 전부 보이는지
- 화면 하단 고정 메뉴가 점검 버튼을 가리더라도 스크롤하면 항목 전체가 위로 올라오는지
- `PDF 저장·공유` 선택 시 인쇄 미리보기에 실제 문서 내용이 표시되는지
