# 작업안전 체크리스트 v1.8.2 업데이트 노트

## 홈 배너 시계 적용

- 홈 배너 우측의 `clipboard.png` 이미지와 원형 체크 표시를 화면에서 제거했습니다.
- Power TBM의 홈 시계 구현을 기준으로 7세그먼트 숫자 매핑과 점멸 콜론을 적용했습니다.
- 현재 시각은 `HH:MM` 형식으로 표시하고 매분 자동 갱신합니다.
- 시계 위에 `7월 25일 토요일` 형식으로 날짜와 요일을 함께 표시합니다.
- 날짜와 시각에는 스크린리더용 한국어 안내 문구를 추가했습니다.

## 동작 및 반응형 처리

- 홈 화면을 벗어날 때 시계 타이머를 정리해 불필요한 반복 실행을 방지합니다.
- 홈으로 돌아오거나 브라우저의 뒤로가기로 화면이 복원되면 시계를 자동 재시작합니다.
- 320px, 390px 스마트폰과 태블릿 화면에서 제목·날짜·시계가 겹치거나 가로로 넘치지 않도록 크기를 조정했습니다.
- 기존 세 가지 메뉴 이미지, 문서 저장 구조, PDF 기능, 96종 체크리스트 데이터는 변경하지 않았습니다.
